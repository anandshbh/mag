Linux 7.0 Lands—Rust Enters the Kernel Mainstream

Linus Torvalds shipped Linux 7.0 on April 12, marking the first major version bump since the 6.x series began. The release cements Rust as a first-class language for kernel development, introduces a revamped process scheduler, self-healing XFS, and broadened support for ARM, RISC-V, and AMD EPYC 5 KVM hypervisors. Ubuntu 26.04 LTS “Resolute Raccoon” will ship it as default on April 23; Arch and openSUSE Tumbleweed packages are already rolling.

Model Context Protocol Crosses 97 Million Installs as Agentic Foundation Takes Shape

Anthropic’s Model Context Protocol (MCP) surpassed 97 million installs in March 2026, cementing its position as the lingua franca of tool-using AI agents. The Agentic AI Foundation, formed under the Linux Foundation in December 2025, anchors the specification alongside OpenAI’s AGENTS.md and Block’s goose framework. Together they form an interoperability layer allowing agents built on different models to share context, invoke tools, and hand off tasks across organizational boundaries—an infrastructure layer the industry has needed since multi-agent workflows became production reality. The March milestone follows explosive adoption among IDE plugins, CI/CD integrations, and enterprise automation platforms now embedding MCP as a first-class interface standard.

Windows 11 April Update Intercepts .RDP File Phishing Campaigns

Microsoft’s April 2026 cumulative update for Windows 11 introduces runtime warnings whenever an .rdp file sourced from an external or untrusted location attempts to execute. The change directly targets a rising wave of phishing campaigns that weaponize Remote Desktop configuration files to achieve lateral movement. The update also removes the long-standing restriction requiring a clean reinstall to disable Smart App Control.
